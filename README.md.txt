# Nadia Preitner Genetic Counselling Website

Static website built for Cloudflare Pages.

## Deployment

1. Upload files to GitHub repository
2. Connect repo to Cloudflare Pages
3. Build settings:
   - Framework: None
   - Output folder: /
4. Deploy

## Features
- Responsive healthcare design
- Multi-page layout
- Contact form (frontend only)
- SEO optimized
